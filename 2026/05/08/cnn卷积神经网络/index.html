CNN卷积神经网络全面总结

一、总结

本文围绕卷积神经网络（CNN） 展开，先讲解全连接神经网络的结构、激活函数（Sigmoid/Tanh/ReLU/Leaky ReLU）、前向传播、损失函数、梯度下降与反向传播的完整计算流程，再指出全连接网络处理图像的缺陷，进而介绍卷积运算、填充、步幅、多通道卷积、池化层的核心原理与尺寸计算公式，最终梳理出CNN 标准结构：输入层→卷积层→池化层→全连接层→输出层。

三、详细总结

3.1 全连接神经网络基础

3.1.1整体结构与单元

结构：输入层→隐藏层→输出层，层间神经元（ 从基础单元到整体结构 ）全连接
神经元数学表达式：a=h(w*x+b) 【对输入做线性变换 + 非线性激活，是所有层的基础。 】
激活函数作用：打破线性组合，让网络可拟合复杂规律

3.1.2常用激活函数对比（图+公式+导数）【神经网络非线性的关键】

加入激活函数的原因：若是考虑把线性函数ℎ(𝑥) = 𝑐𝑥作为激活函数，把𝑦(𝑥) = ℎ(ℎ(ℎ(𝑥)))的运算对应3层神经网络，这个运算会进行𝑦(𝑥) = 𝑐 ∗ 𝑐 ∗ 𝑐 ∗ 𝑥的乘法运算，但是同样的处理可以由𝑦(𝑥) = 𝑎𝑥（注意，这里𝑎 = 𝑐 ∗∗ 3）一次乘法运算（既没有隐藏层的神经网络）来表示

激活函数 公式 优点 缺点
Sigmoid 简单，适合分类 梯度消失、输出非 0 对称
Tanh 0 对称、收敛更快 仍有梯度消失
ReLU 无梯度消失、计算快 神经元死亡
Leaky ReLU 解决神经元死亡 正负区间预测不一致

3.2 图像与全连接网络缺陷

  1. 图像本质：计算机中为数值矩阵，彩色图为 R/G/B 三通道矩阵，灰度图片是单通道

  2. 全连接缺陷：处理图像时会将图像展成一维向量，会丢失空间结构；同时参数量爆炸

3.3. 卷积神经网络

总的来说有卷积运算的就是卷积神经网络，一般流程是输入层→卷积层→池化层（卷积和池化可重复堆叠）→全连接层→输出层，而在卷积和池化操作的过程中一般都是通过卷积核增加通道数，池化减小像素度

输入图像把图片变成数字矩阵（灰度图 / 彩色三通道）。
卷积层（提取特征）用卷积核滑动计算，提取边缘、纹理、形状等特征，输出特征图。
非线性激活加入 ReLU 等激活，让网络能学复杂规律。
池化层（下采样）压缩特征图尺寸，减少计算量，保留关键特征。
重复卷积 + 激活 + 池化层层提取更高级的特征（从局部→整体）。
展平（Flatten）把二维特征图拉成一维向量。
全连接层对高级特征做分类 / 回归判断。
输出层用线性 / Softmax 输出最终结果（分类概率 / 坐标值）。

3.3.1卷积层

1.基础运算
卷积运算：卷积核与输入区域逐元素相乘再求和
带偏置卷积：卷积结果 + 偏置值得到最终输出

2.关键参数与尺寸计算
填充（P）：边缘补 0，控制输出尺寸
步幅（S）：步长
输出尺寸公式：

案例：H=3、P=1、FH=2、S=1 → OH=4

3.多通道卷积
输入通道数 = 卷积核通道数，逐通道卷积后求和得到输出特征图，最终特征图的通道数取决于卷积核的个数

3.3.2 池化层核心原理

作用：减小像素度、保留关键特征、减少计算量，不会使通道数改变
类型：
最大池化：取区域内最大值
平均池化：取区域内平均值

尺寸公式：与卷积层完全一致

四、关键问题与答案

问题1： 回归问题计算损失函数用的均方误差为什么要除以2
方便求导

问题2： 为什么要进行非线性激活，线性激活什么时候用
线性激活：恒等变换，y=x，输出什么就返回什么，作用就是输出连续，无范围限制的真是数值，比如坐标、距离、角度、预测值等
例如在YOLO 最后一层用的就是线性激活（恒等映射、不做非线性压缩）。

非线性激活：
如果只用线性激活，无论网络多少层，最终都等价于一层线性变换，只能拟合直线 / 平面，学不到图像、语音里复杂的非线性规律。非线性激活才能让网络具备深层拟合能力。

问题3： 卷积输出特征图的尺寸由哪些参数决定？计算公式是什么？
由输入尺寸 (H/W)、填充 (P)、卷积核尺寸 (FH/FW)、步幅 (S) 决定

问题4： CNN 相比全连接网络更适合处理图像的核心优势是什么？
CNN 通过卷积提取空间特征、池化降维，大幅减少参数量，同时保留图像的空间位置信息，适配图像数据特性。
